# knn-.-heart-disease-pred
Heart Disease Prediction using K-Nearest Neighbors (KNN) machine learning algorithm with data preprocessing, visualization, and model evaluation.
 Heart Disease Prediction Using KNN

## Project Overview
This project uses the K-Nearest Neighbors (KNN) algorithm to predict whether a patient is likely to have heart disease based on medical data.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Heart Disease Dataset

## Project Steps
1. Data Collection
2. Data Cleaning
3. Data Preprocessing
4. Model Training using KNN
5. Model Evaluation
6. Data Visualization

## Results
The KNN model was trained and evaluated to predict heart disease risk.
